Installation

1. First you need to log into your back office and click on modules page

2. Here you can find option called add a new module

 

3. After you click on add a new module small window will show up. 

4. Click on choose a file and find the module zip (finalmenu.zip) and then click on upload this module. 

5. After the upload is finished you will find the module in modules list. You can filter it by choosing autor: marpaweb.eu.

In desktop menu you can create tabs by clicking on add new tab.

After clicking on add new tab new settings page will show up.

To add new tab click on add button located at the list top right corner. Then you will be redirect to tab creation page.

This page allows to customize each menu tab to fit your needs. In this page you need to first specify the tab name which is required and then you can continue with customizing your tab. Another important thing to note is that you can create tab with simple tab view or advance tab view.

 

Important

Unfortunately it is not possible to use your Horizontal menu in displayTop and displayNav hooks at the same time, since all your content will be duplicated. Please always use one hook at the time. If you don't understand this warning just ignor it.

BLOCK UPDATE BUTTON

When you press the pen button and specific block options will show up you need to be of aware that you have to press the update button after you are done with the block settings to update object model before you can hide the block options and continue to next one or press save.

Or whenever you are making some updates in specific block you need to after the editation is finished press the update button to update object model before you can save your changes. Otherwise your changes will not be saved and you will lose your editation.
