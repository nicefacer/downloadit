Installation

Installation

Go to your prstashop admin and open module section.


Now click on “ad new module”


From your local, select your shortcode.zip file and upload it.

 

How to use this module:

How to use this module:

Your Smartshortcodemodule has three basic functional parts.

By using number 2 and 3optins, you can use your shortcode features at anywhere of your site.Number 1 Shortcode set is also available at your CMS pages` editing end.It is represented with "D" letter at text editing box options.

  1. A full set of short-code, where ere after clicking on each button,relevant option and settings will appear.

  2. A complete content- anywhere building system where number1 option is also available.
  3. A product tab building system where number 1 option is also available.

Content anywhere

Using Content anywhere to build any kind of content for prestashop hooks, categories and products:

From your admin>shortcode, select content anywhere option.

You can display any kind of HTML markup with text, audio, video, images, banner, sliders and others at any available hook of your prestashop.

You can attach your content to any specific category.

And even to any specific products.

You can create such contents in unlimited numbers.

And our full set of shortcode is also available to create your contents anywhere!!!!!

Product tab

Using unlimited product tab to add any kind of content/description for products:

From your admin>shortcode, select product tab creator option.

You can create such contents in unlimited numbers.

You can create separate tab for each product or a common tab for all of your products.

You can display any kind of HTML markup with text, audio, video, images, banner, sliders and our full set of shortcode is also available to create your product tab contents!!!!!

Q I want to show the shortcode element in Shortcode module is not installed properly; gives  unable to install override:  message?
 
Ans If after installing your module, you are getting &ldquo;unable to install override like message, it may be due to following two causes.

1. Inside of your

root>prestashop>cache

location, there is file named class_index.php Rename this file and try to install your module again.

2.Inside of your

smartshortcode>override>controller>front

, there is a file

cmscontroller.php

Copy this file and replace this at

root>prestashop>override>controllers>front

location.Then install your module.
